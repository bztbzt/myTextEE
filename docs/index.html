TextEE: A Standardized, Fair, and Reproducible Benchmark for Event Extraction

Kuan-Hao Huang1, I-Hung Hsu2, Tanmay Parekh3, Zhiyu Xie4, Zixuan Zhang1, Premkumar Natarajan2, Kai-Wei Chang3, Nanyun Peng3, Heng Ji1
1University of Illinois Urbana-Champaign, 2University of Southern California, 3University of California, Los Angeles, 4Standford University
Paper arXiv Code

Introduction

TextEE is a standardized, fair, and reproducible benchmark for evaluating event extraction approaches.

  • Standardized data preprocessing for 10+ datasets.
  • Standardized data splits for reducing performance variance.
  • 10+ implemented event extraction approaches published in recent years.
  • Comprehensive reevaluation results for future references.

We will keep adding new datasets and new models!

News

  • [2023/11/15] We release TextEE. Feel free to contact us if you want to contribute your models or datasets!

Supported Datasets

  • ACE05: The Automatic Content Extraction (ACE) Program - Tasks, Data, and Evaluation, LREC 2004
  • RichERE: From Light to Rich ERE: Annotation of Entities, Relations, and Events, EVENTS@NAACL 2015
  • M2E2: Cross-media Structured Common Space for Multimedia Event Extraction, ACL 2020
  • CASIE: CASIE: Extracting Cybersecurity Event Information from Text, AAAI 2020
  • PHEE: PHEE: A Dataset for Pharmacovigilance Event Extraction from Text, EMNLP 2022
  • MEE: MEE: A Novel Multilingual Event Extraction Dataset, EMNLP 2022
  • FewEvent: Meta-Learning with Dynamic-Memory-Based Prototypical Network for Few-Shot Event Detection, WSDM 2020
  • MAVEN: MAVEN: A Massive General Domain Event Detection Dataset, EMNLP 2020
  • RAMS: Multi-Sentence Argument Linking, ACL 2020
  • WikiEvents: Document-Level Event Argument Extraction by Conditional Generation, NAACL 2021
  • GENEVA: GENEVA: Benchmarking Generalizability for Event Argument Extraction with Hundreds of Event Types and Argument Roles, ACL 2023

Supported Models

  • DyGIE++: Entity, Relation, and Event Extraction with Contextualized Span Representations, EMNLP 2019
  • OneIE: A Joint Neural Model for Information Extraction with Global Features, ACL 2020
  • AMR-IE: Abstract Meaning Representation Guided Graph Encoding and Decoding for Joint Information Extraction, NAACL 2021
  • DEGREE: DEGREE: A Data-Efficient Generation-Based Event Extraction Model, NAACL 2022
  • EEQA: Event Extraction by Answering (Almost) Natural Questions, EMNLP 2020
  • RCEE: Event Extraction as Machine Reading Comprehension, EMNLP 2020
  • Query&Extract: Query and Extract: Refining Event Extraction as Type-oriented Binary Decoding, ACL-Findings 2022
  • TagPrime: TAGPRIME: A Unified Framework for Relational Structure Extraction, ACL 2023
  • UniST: Unified Semantic Typing with Meaningful Label Inference, NAACL 2022
  • BART-Gen: Document-Level Event Argument Extraction by Conditional Generation, NAACL 2021
  • PAIE: Prompt for Extraction? PAIE: Prompting Argument Interaction for Event Argument Extraction, ACL 2022
  • X-Gear: Multilingual Generative Language Models for Zero-Shot Cross-Lingual Event Argument Extraction, ACL 2022
  • AMPERE: AMPERE: AMR-Aware Prefix for Generation-Based Event Argument Extraction Model, ACL 2023
